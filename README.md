### 后台管理系统


 **项目基于:.Net Core 2.1 
运行环境：vs2017 .Net Core 2.1 

(请确保安装了 SDK 2.1.301 下载地址：http://www.microsoft.com/net/download/dotnet-core/2.1#sdk-2.1.300-preview1）** 


数据库在 wwwroot/AppDB 文件夹下面 

 **_交流群: 534584927_** 

前端： BootStrap 3 、 Layer 、 Jquery 后端： 三层+ .NetCore 2.1 MVC
